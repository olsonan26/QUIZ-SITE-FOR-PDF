# QUIZ SITE FOR PDF

Behavioral typing quiz funnel for parents — "The Day Your Child Was Born"

## Live Site
https://thedayofbirth-88b32b01.viktor.space

## Stack
- Vite + React + TypeScript
- Framer Motion (animations)
- Convex (real-time backend)
- Tailwind CSS
- SamCart (checkout)

## Key Files
- `src/App.tsx` — Main quiz application (landing, quiz, results, offer screens)
- `src/data.ts` — Quiz questions, type definitions, result descriptions
- `convex/` — Backend (leads, quiz results storage)

## Setup
```bash
bun install
bun run dev
```
